# Amazon Clone - College Project

Simple Amazon homepage clone made with HTML, CSS and basic JavaScript.

## Pages
- index.html - Home page with banner slider and product cards
- product.html - Product detail page (opens when you click a product)
- deals.html - Today's deals listing
- fashion.html - Fashion category page
- electronics.html - Electronics category page
- gaming.html - Gaming category page
- cart.html - Shopping cart
- signin.html - Sign in form
- orders.html - Your orders page
- help.html - Customer service / help page

## How to run
Open index.html in browser or use Live Server extension.

## JavaScript features
- Banner slider (manual + auto)
- Add to cart using localStorage
- Cart count updates on all pages
- Product page loads from URL id
- Basic sign in form validation
- Search box with alert
